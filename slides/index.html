The Prediction Model

This prediction model takes two teams as input and will output the predicted winner. The prediction model is trained with daily ESPN BPI data and a recent performance indicator.

The recent performance indicator takes the most recent BPI for a given team and divides it by the average BPI for that team over a given time period. A larger and more positive value indicates a team that has recently been performing well, while a larger and more negative value indicates a team that has recently been performing poorly.

The Data

A quick look at the training data...

bpi %>% filter(BPI.RK <= 351) %>% group_by(TEAM) %>%
select(DATE, TEAM, BPI.RK, BPI) %>%
mutate(recentPerf=(BPI-mean(BPI))/mean(BPI)*100) %>%
filter(DATE==as.character(Sys.Date())) %>% group_by() %>% arrange(desc(recentPerf))
