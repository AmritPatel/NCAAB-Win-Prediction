The Prediction Model

This prediction model takes two teams as input and will output the predicted winner. The prediction model is trained with daily ESPN BPI data and a recent performance indicator.

The recent performance indicator first takes the most recent BPI for a given team, then subtracts the average BPI over time for the team, and finally divides that quantity by the average BPI over time for the team. A larger and more positive value indicates a team that has recently been performing well, while a larger and more negative value indicates a team that has recently been performing poorly.
